# mystore 果蔬微店
#### 项目简介

大二下学期课程设计 果蔬微店-JavaWeb

一个基于 Tomcat、JavaWeb、JSP、 MySQL、c3p0



#### 系统功能

- 用户管理：用户CRUD和分页查询

- 商品管理：对商品的价钱图片简介等CRUD

- 订单管理：查看订单

  

#### 项目结构

项目采用按功能分模块开发方式

- WebContent

  - admin 商品浏览下单

  - manager 后台管理
